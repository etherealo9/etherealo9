## `$ whoami`

I'm Arghyadeep. I started in marketing, detoured into game dev on Unreal Engine, and landed in full-stack engineering -- the scenic route.

I studied English literature at the University of Calcutta, then shifted to an MCA in AI/ML at MAKAUT. I also did a Product Space Fellowship, which gave me a proper framework for product thinking. But most of what I know, I learned by shipping things and breaking them at 3 AM.

I think across fields because problems don't care about department labels. I read philosophy and quantum information theory -- not because a syllabus told me to, but because you genuinely cannot express quantum information in classical math. I build products from user intent to deployment. I care about system design, cloud architecture, and front-end that actually feels good to use. I write code for real people, not for resumes.

Currently at Amazon. Previously co-founded a few things. Quietly drawn to biotech. Always building something.

## `$ ls /projects`

**[Bmatch.in](https://bmatch.in)** -- VC matching platform. Upload a pitch deck, the AI reads it, scores the financials, and matches you with aligned investors. Built on Next.js and Python, powered by FinsecAGI for automated model validation and compliance scoring.

**[MindSync](https://github.com/etherealo9/MindSync)** | [npm](https://www.npmjs.com/package/@mindsync/cli) -- Privacy-first AI journaling where your API key never leaves your machine. BYOK architecture means no server ever sees your entries. Available as both web app and CLI tool for terminal users.

**[CTK Duration Calculator](https://ctk-timecalculator.vercel.app)** -- Keyboard-first time calculation tool built at Amazon for UK/US payroll teams processing hundreds of entries daily. Modern React/TypeScript/Vite stack with Docker containerization. Reduced manual processing time by 70%.

**[FinsecAGI](https://github.com/etherealo9/FinsecAGI)** *(private)* -- Financial ML engine built as a 6-agent LangGraph orchestration system. Handles automated pitch deck evaluation, risk scoring, and investor-startup thesis alignment. The intelligence backbone powering Bmatch.in backend.

**[SearchAlgoGym](https://github.com/etherealo9/searchalgogym)** *(private)* -- Quantum-inspired algorithm research playground. ARDP Gen 2 explores search complexity past classical ceilings using TypeScript and Python. Benchmarks and theoretical frameworks included.

**[Isochron](https://github.com/etherealo9/Isochron)** *(private)* -- Focus chess clock for deep work. React Native + Expo mobile app with Zustand state management and Firebase Firestore. Neo-brutalist UI with mechanical metaphor for attention management. DND integration, offline-first, haptic feedback.

**[3sGroomers](https://3sgroomers.com)** -- VR soft-skills training platform where I was CPO. Shipped MVP to 500+ users. Integrated Meta Quest and Vive hardware with cloud backend. Built on Unity and Unreal Engine architecture.
